# API Ilotel

## Prerequisites:

- [Vercel CLI](https://vercel.com/docs/cli) installed globally

To develop locally:

```
npm install
vc dev
```

```
open http://localhost:3000
```

To build locally:

```
npm install
vc build
```

To deploy:

```
npm install
vc deploy
```
## Contry / Region code :

Contry use **ISO 3166-1 alpha-2**

Region use specific code :

| Region       | Code    |
| ------------ | ------- |
| Europe       | `eu`    |
| Asia         | `as`    |
| Africa       | `af`    |
| America      | `am`    |
| Oceania      | `oc`    |
| Middle East  | `me`    |
| World        | `ww`    |
